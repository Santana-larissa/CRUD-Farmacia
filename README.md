## 🏥 CRUD-Farmácia — API Restful com NestJS e PostgreSQL

Este projeto é uma API backend desenvolvida com NestJS, que simula o sistema de uma farmácia. O objetivo é fornecer uma estrutura sólida para gerenciamento de produtos farmacêuticos, classificados por categorias, com foco em organização, escalabilidade e boas práticas de desenvolvimento.

### ✨ Funcionalidades

✅ CRUD completo de Categorias  
✅ CRUD completo de Produtos, com relacionamento com Categorias  
✅ Validações e regras de negócio implementadas  
✅ Conexão com banco de dados PostgreSQL usando TypeORM  
✅ Estrutura de projeto baseada em boas práticas (modularização, versionamento por branches, etc.)

### 🛠️ Tecnologias Utilizadas

- NestJS  
- TypeScript  
- PostgreSQL  
- TypeORM  
- Node.js  
- Insomnia/Postman para testes de API  

### 📂 Organização do Projeto

O projeto é dividido em branches, cada uma representando uma etapa do desenvolvimento:

- `Configurando-o-Projeto`: estrutura inicial do projeto + banco de dados  
- `CRUD-Categoria`: implementação do CRUD de Categoria  
- `CRUD-Produto`: implementação do CRUD de Produto com relacionamento  

### 🚀 Como Executar

1. Clone o repositório  
2. Instale as dependências com `npm install`  
3. Configure seu banco de dados PostgreSQL  
4. Rode o projeto com `npm run start:dev`
